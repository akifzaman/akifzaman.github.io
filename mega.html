PluginMega for Unity

Unity plugin for mega

The Cloud File Storage 23 - Mega plugin is designed to make document storage easier and more efficient for Unity game engine users. By integrating the plugin into Unity, users can easily store documents without leaving the engine. This plugin was created using the MegaApIClient package. Unitask also in assistance creating this plugin.

The plugin is compatible with Windows, Android, and iOS, making it accessible to a wide range of users.

Key Features

  • Streamlined document storage
  • No need to browse MEGA in a browser
  • Compatible with multiple platforms
  • Easy integration with the Unity game engine
  • Suitable for personal and app use
