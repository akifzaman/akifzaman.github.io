Augmented RealityTile TryOn

place tiles on floors and walls in AR

This Unity AR project empowers users to place tiles seamlessly on both floors and walls, offering a versatile environment customization experience. While AR Foundation provides robust horizontal plane detection, its vertical plane detection lacks consistency, especially on Android devices. To overcome this limitation, a custom solution was implemented: users can manually mark points on walls, and the application generates a mesh in real-time to fit the selected area. This custom mesh approach not only ensures precise placement but also supports dynamic tiling and texture application on vertical surfaces, enhancing the overall functionality and user experience.

Key Features

  • Dual-Surface Placement
  • Custom Vertical Plane Solution
  • Dynamic Mesh Generation
  • Texture and Tiling Support
  • Enhanced User Experience
  • Device Agnostic
